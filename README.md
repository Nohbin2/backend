🔐 Auth Module – Contributor README (feature/auth)
담당 범위

본인은 본 프로젝트에서 feature/auth 브랜치의 인증(Auth) 기능 개발을 담당했습니다.
카카오 OAuth 로그인을 기반으로 채팅 서비스에서 사용할 사용자 인증 흐름을 구현하는 것이 주요 역할이었습니다.

✨ 구현 기능 (Auth 파트)

카카오 OAuth 2.0 로그인 연동

카카오 로그인 API를 활용한 사용자 인증 처리

별도 회원가입 없이 카카오 계정을 통한 로그인 기능 구현

사용자 정보 수집 및 처리

로그인 성공 시 카카오로부터 사용자 이름 및 프로필 이미지 정보 수신

채팅 서비스에서 사용할 사용자 식별 정보로 가공 및 저장

인증 기반 사용자 식별 구조 설계

인증된 사용자만 채팅 기능에 접근할 수 있도록 인증 흐름 설계

이후 채팅/알림 기능과 연동 가능한 인증 구조 제공

백엔드 인증 API 구현

프론트엔드와 연동되는 인증 관련 API 엔드포인트 개발

OAuth 인증 흐름에 맞춘 요청/응답 구조 설계

🛠 사용 기술

Authentication: Kakao OAuth 2.0

Backend: Spring Boot

API: RESTful API 기반 인증 엔드포인트

Collaboration: GitHub Flow 기반 브랜치 개발 (feature/auth)

📌 기여 요약 (포트폴리오용)

카카오 OAuth 2.0 기반 로그인 기능을 담당하여,
실시간 채팅 서비스의 사용자 인증(feature/auth) 백엔드 로직을 설계·구현했습니다.

📎 참고

본 기여 내용은 feature/auth 브랜치 기준이며,
다른 기능 구현은 다른 팀원이 담당했습니다.

팀 프로젝트 특성상 역할을 명확히 분리하여 개발했습니다.
